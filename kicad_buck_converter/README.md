# Понижающий DC-DC преобразователь на LM2596S-5.0

## Описание проекта
Проект понижающего (buck) DC-DC преобразователя для KiCad 10 на базе популярной микросхемы LM2596S-5.0.

## Характеристики
- **Входное напряжение**: 4.5V - 40V (рекомендуется 12V)
- **Выходное напряжение**: 5V (фиксированное)
- **Максимальный ток**: 3A
- **КПД**: до 88%
- **Частота переключения**: 150 kHz

## Компоненты
| Обозначение | Номинал | Описание | Footprint |
|-------------|---------|----------|-----------|
| U1 | LM2596S-5.0 | Понижающий регулятор | TO-263-5 |
| C1 | 10uF | Входной конденсатор | CP_Radial_D5.0mm_P2.50mm |
| C2 | 22uF | Выходной конденсатор | CP_Radial_D5.0mm_P2.50mm |
| L1 | 10uH | Дроссель | L_Radial_D5.0mm_P2.50mm |
| D1 | SS34 | Диод Шоттки | DO-214AC_SMA_Horizontal |
| J1 | - | Входной клеммник | TerminalBlock_Phoenix_MKDSN-2-5.08 |
| J2 | - | Выходной клеммник | TerminalBlock_Phoenix_MKDSN-2-5.08 |

## Как открыть проект
1. Установите KiCad 10 (или последнюю доступную версию)
2. Откройте KiCad
3. Файл → Открыть проект
4. Выберите файл `buck_converter.kicad_sch` в папке `/workspace/kicad_buck_converter/`
5. Для просмотра PCB откройте `buck_converter.kicad_pcb`

## Структура файлов
- `buck_converter.kicad_sch` - Электрическая схема
- `buck_converter.kicad_pcb` - Печатная плата
- `README.md` - Этот файл

## Схема подключения
```
Вход (J1) → C1 → U1 (VIN) → U1 (SW) → L1 → C2 → Выход (J2)
                    ↓           ↓
                   GND        D1 (катод к SW, анод к GND)
```

## Рекомендации по трассировке
1. Сделайте дорожки питания широковыми (минимум 1мм для тока 3A)
2. Разместите компоненты компактно для минимизации паразитных индуктивностей
3. Используйте полигон земли на нижнем слое
4. Входной конденсатор C1 должен быть максимально близко к выводам VIN и GND микросхемы

## Примечания
- Проект создан для сквозного монтажа (THT) компонентов, кроме микросхемы и диода (SMD)
- Для версии с регулируемым выходным напряжением используйте LM2596S-ADJ и добавьте делитель напряжения
- Убедитесь, что все компоненты рассчитаны на необходимое напряжение и ток
