# Buck DC-DC Converter (LM2596S-5.0)

## Описание проекта

Проект понижающего (step-down) DC-DC преобразователя на базе микросхемы **LM2596S-5.0** для KiCad 10.

### Характеристики:
- **Входное напряжение:** 4.5V - 40V (рекомендуется 12V)
- **Выходное напряжение:** 5V (фиксированное)
- **Максимальный выходной ток:** 3A (рекомендуется до 2A)
- **Частота переключения:** 150 kHz
- **КПД:** до 88%

## Состав проекта

### Файлы:
- `buck_converter.kicad_sch` - электрическая схема
- `buck_converter.kicad_pcb` - печатная плата
- `README.md` - документация

## Компоненты

| Обозначение | Наименование | Номинал | Корпус |
|-------------|--------------|---------|--------|
| U1 | LM2596S-5.0 | Buck Regulator | TO-263-5 |
| C1 | Конденсатор | 10uF/50V | 0805 |
| C2 | Конденсатор | 22uF/16V | 1206 |
| L1 | Дроссель | 10uH/3A | 1210 |
| D1 | Диод Шоттки | SS34 (3A/40V) | SMA |
| R1 | Резистор | 1kΩ | 0805 |
| J1 | Клемма | Вход | Phoenix MKDSN 2.5/2 |
| J2 | Клемма | Выход | Phoenix MKDSN 2.5/2 |

## Схема подключения

```
VIN (12V) ──┬── C1 ──┬── U1 (VIN)
            │        │
           GND      U1 (SW) ──┬── L1 ──┬── VOUT (5V)
                              │        │
                             D1       C2
                              │        │
                             GND      GND
```

## Инструкция по использованию

1. Откройте KiCad 10
2. Выберите "File" → "Open Project"
3. Найдите и откройте файл `buck_converter.kicad_pro` (или создайте новый проект и импортируйте файлы)
4. Для редактирования схемы используйте **Schematic Editor**
5. Для редактирования платы используйте **PCB Editor**

## Рекомендации по трассировке

- **Силовые цепи:** Трассы VIN, SW и VOUT должны быть максимально широкими (минимум 1.5mm)
- **Земля:** Используйте сплошной полигон земли на нижнем слое
- **Терморазгрузка:** Убедитесь, что вывод GND микросхемы U1 имеет хорошую тепловую связь с полигоном земли
- **Компоненты:** Размещайте C1 как можно ближе к выводам VIN и GND микросхемы

## Меры предосторожности

⚠️ **Внимание!** При работе с напряжениями выше 24V соблюдайте меры безопасности:
- Не прикасайтесь к плате под напряжением
- Используйте соответствующие средства защиты
- Проверяйте полярность подключения перед подачей питания
