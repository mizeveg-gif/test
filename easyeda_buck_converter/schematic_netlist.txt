# EasyEDA Buck Converter Project Files

## 📁 Содержимое папки

Этот проект содержит все необходимые файлы и инструкции для создания понижающего DC-DC преобразователя в EasyEDA.

### Файлы:
- `README_RU.md` - Полная документация на русском языке
- `project_info.json` - Структурированная информация о проекте (компоненты, соединения, параметры)
- `schematic_netlist.txt` - Текстовое описание схемы для импорта

## 🚀 Быстрый старт

### Вариант 1: Ручное создание (рекомендуется)
1. Откройте [EasyEDA](https://easyeda.com)
2. Создайте новый проект: **File → New → Schematic**
3. Следуйте пошаговой инструкции в `README_RU.md`

### Вариант 2: Импорт через LCSC
Компоненты можно найти по LCSC номерам из файла `project_info.json`:
- U1: C2891 (LM2596S-5.0)
- C1, C2: C134976 (22uF электролит)
- L1: C2883654 (10uH дроссель)
- D1: C130756 (SS34 диод Шоттки)
- J1, J2: C2380036 (клеммы 5mm)

## 🔌 Назначение выводов LM2596S-5.0

| Вывод | Название | Описание | Подключение |
|-------|----------|----------|-------------|
| 1 | VIN | Входное напряжение | +Вход, C1+ |
| 2 | OUTPUT | Выход ключа | L1, D1(анод) |
| 3 | GND | Общий провод | GND схемы |
| 4 | FEEDBACK | Обратная связь | Не подключать (для фикс. 5V) |
| 5 | ON/OFF | Включение/выключение | GND (включено) |

## 🏭 Производство PCB

После завершения проекта в EasyEDA:
1. Нажмите **Fabrication → PCB Order**
2. Выберите параметры:
   - Количество: 5-10 шт (минимальный заказ)
   - Размер: ~50x40mm
   - Слоёв: 2
   - Цвет паяльной маски: зелёный (стандарт)
   - Покрытие: HASL (бессвинцовое)
3. Стоимость: ~$5-10 за 5 плат

## 🛠️ Альтернативные компоненты

Если указанные компоненты недоступны, можно использовать аналоги:

### LM2596S-5.0 замены:
- LM2596T-5.0 (корпус TO-220)
- MP2307 (современный аналог)
- XL1509-5.0E

### Диод SS34 замены:
- SS36 (40V → 60V)
- SK34
- B340A

### Дроссель 10uH:
- Любой на ток ≥3A
- Индуктивность 10-22uH

## ⚙️ Настройка на другое выходное напряжение

Для версии **LM2596-ADJ** (регулируемая):

```
        R1
VOUT ──┬───────┐
       │       │
      ┌┴┐      │
      │ │ R2   ├─── Pin 4 (FB)
      └┬┘      │
       │       │
      GND     GND

Формула: Vout = 1.23V × (1 + R1/R2)

Для 5V: R1=3kΩ, R2=1kΩ
Для 3.3V: R1=1.7kΩ, R2=1kΩ
Для 12V: R1=8.8kΩ, R2=1kΩ
```
